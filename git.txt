1. Основные команды
    git init — создать репозиторий (начать работать с Git).
    git add <имя_файла> — добавить файл в список для сохранения.
    git commit -m "Описание" — сохранить изменения с комментарием.
    git status — показать текущее состояние (какие файлы изменены).
    git log — показать историю сохранений.
    git push — отправить свои изменения на сервер.
    git pull — скачать изменения с сервера.
2. Ветвление
    Что такое ветка?
    Это как отдельная дорожка для работы над задачей, чтобы не мешать другим.
    Создать ветку:
        git branch имя_ветки.
    Переключиться на ветку:
        git checkout имя_ветки.
    Слить ветку с основной:
        git merge имя_ветки.
3. Работа в команде
    Скачивать чужие изменения:
        git pull — чтобы всегда быть в курсе.
    Отправлять свои изменения:
        git push — чтобы команда увидела ваш прогресс.
    Если что-то конфликтует (например, вы и коллега поменяли один файл):
    Git попросит вручную исправить файл, а потом написать:
        git add и git commit.
